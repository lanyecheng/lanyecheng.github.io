Algolia 搜索服务

注册 Algolia
注册一个新账户。可以使用 GitHub 或者 Google 账户直接登录，注册后的 14 天内拥有所有功能（包括收费类别的）。之后若未续费会自动降级为免费账户，免费账户 总共有 10,000 条记录，每月有 100,000 的可以操作数。注册完成后，创建一个新的 Index，这个 Index 将在后面使用

点击 API Keys 进入页面，点击 All API Keys 选择右上方的 New API Key 新建一个用于管理 Index 的 API Key，Description 随意，Indices 添加上面创建的 Index，ACL 中只选择四种操作 addObject、deleteObject、deleteIndex、listIndexes，其它的默认就行，然后点击 Create

在站点根目录下安装插件 hexo-algolia
